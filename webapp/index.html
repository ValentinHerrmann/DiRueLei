Daten aus Artemis abrufen

Zum Abrufen der Studierendendaten benötigen Sie Ihren JWT-Token aus Artemis.
Wie erhalte ich den JWT-Token?

  1. Melden Sie sich bei artemis.tum.de an.
  2. Öffnen Sie die Entwicklerkonsole (F12) → Reiter Application (Chrome) bzw. Storage (Firefox).
  3. Unter Cookies → https://artemis.tum.de finden Sie den Cookie jwt.
  4. Kopieren Sie den Wert (langer Text) und fügen Sie ihn unten ein.
